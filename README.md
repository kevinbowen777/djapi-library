## djapi-library

Basic library website & API built with Django & Djano REST Framework (DRF)

### Installation
 - `git clone https://github.com/kevinbowen777/djapi-library.git`
 - `cd djapi-library`
 - `poetry shell`
 - `python manage.py runserver`
 - Open browser to http://127.0.0.1:8000`
